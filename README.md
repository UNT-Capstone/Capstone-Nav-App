## Project Overview

The UNT Navigation App helps users:
- Navigate campus using interactive maps  
- Get real-time routes and directions  
- Manage personal schedules  
- Discover and create campus events  
- Connect with other users socially
  
## Tech Stack
### Frontend
- React  
- TypeScript  
- Next.js  

### Backend & Database
